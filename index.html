JQuery Hashchange

This plugin was inspired and forked from Andrey Popelo. Thanks to Andrey popelo.

Features:

  • Calls onSet() function when the window.loction.hash sets.
  • Calls onRemove() function when the window.loction.hash removed.
  • Support the dynamic hashes i.e. fragment values which are generated dynamically. example: #user/123 - To view the details of user with id 123. #user/123/edit - To edit the user details with id 123
  • Passing values of dynamic hashes to the onSet and onRemove functions.

Example:

  • Here the onSet and onRemove functions are called when the hash "#company" sets or remove respectively. ` $(window).hashchange(

            {

            hash: "#company",

            onSet: function() {

                $("#compList").show();

            },

            onRemove:function(){

                $("#compList").hide();

            }

        }

);

`

  • Setting hash functions (onSet and onRemove) for dynamic hashes. Getting the values in onSet and onRemove functions. For example the url is http://abc.com/company.html#company/123 . Then the 123 dynamic value pass to onSet and onRemove as array.

` // Decalre hash function in array.

$(window).hashchange([

{

hash: "#company/%s",

regex: true,

onSet: function(arr) {

          var companyId=arr[0];

          $("#compDetail").show();

       },

onRemove:function(arr){

          var companyId=arr[0];

          $("#compDetail").hide();

       }

},

{

hash: "#company/%s/emp/%s",

regex: true,

onSet: function(arr) {

          var companyId=arr[0];

          var employeeId=arr[1];

          $("#empDetail").show();

       },

onRemove:function(arr){

          var companyId=arr[0];

          var employeeId=arr[1];

          $("#empDetail").hide();

       }

}

]);

`
